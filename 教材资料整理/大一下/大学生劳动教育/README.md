## 考试说明

与其他慕课考试形式相同，直接答题即可。

平时分需要课堂讨论+结课一次小组短视频/写篇小作文。更推荐写小作文，相比做视频花的时间会好很多，且分数也不低。有条件的话最好多加一些照片。
