## 考试说明

分数构成：平时分+期中考试+期末考试

期末考试为简答题+计算题（4:6），10道题目

期中考试为五道大题目，一小时之内完成

整体还是偏向于考常见题型和常见模型的。



## 课程评价

这课可太难了，大二上学期仍然是相对较难的一门。

电磁学挺难的，模型看起来比较复杂（但实际上看明白了就好了），光学和量子力学倒还好（考的并不会很难）

也可以在考试前看高斯课堂速成的，应付考试来说，模型相对都很固定。

考前几节课老师可能会透露当年的考试范围，一定要听（尤其是最后一节课）
