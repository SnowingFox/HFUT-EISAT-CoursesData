## 考试说明

考试的时间挺后面的，甚至有可能是最后一门考的课，考完概率论才能回家。

分老师情况，但总体平时会有课堂测验。

期末考试的卷子完全可以参考往年，题型，难度都基本一致。





## 课程评价

总的来说还是比较容易的一门课。与线性代数差不多，考试的最后的分数情况也差不多。好好学应该可以拿到一个还不错的分数。
