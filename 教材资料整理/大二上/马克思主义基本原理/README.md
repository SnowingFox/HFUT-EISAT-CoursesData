## **考试说明**

学习通考试，与之前的思修/近代史一致，只有单选+多选+判断，无大题目。



## **课程评价**

分老师，不同老师对作业的要求不同。总体来说除了要写一篇论文一样的类似物，做个PPT以外，相对来说还是比较轻松的课。考前最多两天前刷完一遍学习通题库就好了，不需要在平时花太多时间准备。
