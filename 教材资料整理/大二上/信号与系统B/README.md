## 考试说明

有期中考试（1-3章），也有期末考试（前三章百分之30，后面百分之70）。两次考试中间还有个课堂测试。期末考试之前还会有个实验（Matlab仿真，很简单）

如果不能深入理解的话，记忆一下常见的变换对，熟悉一下常考的集中题型，应该不会有什么大问题。





## 课程评价

挺重要的专业课（然而我翻车了.jpg）。如果感觉上课内容听不懂的话不妨通过PPT学习，或者在B站上查看西电的信号与系统网课学习。个人觉得讲的还是很不错的。
