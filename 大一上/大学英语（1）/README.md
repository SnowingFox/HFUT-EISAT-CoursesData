## 课程评价

是英语课，但并不能学到什么英语知识。所以，也是水课。（取决于老师）

从20级开始，快慢班似乎没有任何区别了，也不存在什么先考四六级啥的。学校都是统一四级大一下考。

口语考试各个老师有所不同。有的老师是配音，有的老师是演讲，甚至有的老师是唱歌~

不过，从备战四六级的角度，学学英语也是好的呀~~~（虽然没学过.jpg）~~
